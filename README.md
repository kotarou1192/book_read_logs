# book_read_logs
books that i have to read/been reading/read.

## ルール

### 本を買ったら
- Projectsの未読に追加

### 本を読み始めたら
- 未読->読書中に移動
- このリポジトリにディレクトリを作る
  - ディレクトリごとにVSCodeのTODO+でリストを使り進捗を管理
- 必要に応じて書いたコードや感想、疑問点を記録

### 本を読み終えたら
- 読書中->既読に移動
